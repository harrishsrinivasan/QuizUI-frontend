# Quiz App

This project is a Vite + React + TypeScript quiz application styled with Tailwind CSS and shadcn-ui components.

## Getting started

```sh
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui

## Project structure

- `src/`: Application source code
- `public/`: Static assets
- `index.html`: HTML entry point
- `vite.config.ts`: Vite configuration

## Notes

- Requires Node.js and npm installed locally.
- Update environment variables and API endpoints as needed for your deployment.
