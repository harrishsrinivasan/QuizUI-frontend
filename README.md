# Quiz App

An interactive quiz application built with Vite, React, and TypeScript, styled with Tailwind CSS and shadcn-ui components.

## Technologies used

- Vite (build + dev server)
- React 18 with TypeScript
- Tailwind CSS (utility styling)
- shadcn-ui (prebuilt UI primitives)
- Radix UI primitives (accessible components)
- React Router (routing)
- React Query (@tanstack/react-query for data fetching/caching)
- ESLint + TypeScript ESLint (linting)

## Directory overview

- `src/` — application source
  - `pages/` — routed pages (main quiz, 404, etc.)
  - `components/` — reusable UI and quiz components
  - `hooks/` — shared React hooks
  - `lib/` — utilities
  - `index.css` — global styles (Tailwind entry)
  - `main.tsx` — React entry point
- `public/` — static assets (favicons, robots.txt, etc.)
- `index.html` — HTML entry for Vite
- `vite.config.ts` — Vite configuration
- `tailwind.config.ts` — Tailwind configuration
- `tsconfig*.json` — TypeScript configs
- `eslint.config.js` — lint configuration

## How to run

Prerequisites: Node.js (LTS recommended) and npm.

```sh
# Install dependencies
npm install

# Start the dev server with hot reload
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview

# Lint the codebase
npm run lint
```
